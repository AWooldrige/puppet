Facebook Messenger browser not supported

Unfortunately, Facebook Messenger mobile browser does not support the login mechanism used to secure photos.wooldrige.co.uk from the public (see this bug report).

Either:

  • Use a laptop or desktop computer to visit link. It's only when using Facebook Messenger on a mobile device that causes a problem.
  • Copy the URL, then open your normal mobile web browser and paste the URL into the address bar.
  • Configure Facebook Messenger to open this link in an external browser See this guide.
